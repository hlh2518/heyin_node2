1、requirements.txt中打开了jpype1（使用jar必备插件）
2、安装部署；
  1）git clone https://ghfast.top/https://github.com/hlh2518/heyin_node2.git
  2) mv heyin_node2 heyin && cd heyin
  3) chmod +x hy.sh hy_install.sh
  4) ./hy_install.sh
  5) cd heyin_parser
  6) npm install
  7) cd ..
  8) cp -r hy.sh /hoem/heyin
  9) cd hlhwan
  10) ./hy.sh
